F# and WCF Data Contracts

Ted Neward recently wrote about using F# to write WCF services. I share his enthusiasm; so much so that Iâ€™m using F# to write examples as I study for MCTS 70-503.

Good examples are hard to find. Most of the time books will give incomplete, or worse, erroneous examples. I have striven to boil each example down to just the essentials needed to get it to run. The examples use F# Interactive and each example is a fully encapsulated, self-sufficient demonstration of some WCF concept. To run an example, simply type fsi --exec <script name>. You can see my examples here: https://bitbucket.org/rvernagus/wcf-examples

I am planning to run a series of posts based on my experience in using F# to write WCF services. Stay tuned for that!
